# 当前工作版本说明

## 用途

`docs/report/current/` 用于存放当前正在维护的报告工作版本。

## 当前状态

目前这里承载的是 `v0.2` 的工作版本，即在 `v0.1` 冻结版基础上的新一轮持续修订稿。

## 工作规则

1. 新的正文写作、章节扩展、引用补充优先在本目录中进行。
2. 当某一版本完成并确认后，应将稳定内容整理到 `docs/report/versions/<version>/`。
3. 后续版本仍建议先在 `current/` 工作，再冻结到 `docs/report/versions/<version>/`。
4. 当前阶段以“技术主干深度、工程案例、图表资产与研究卡片联动”为主。

## 配套目录

- `figures/`：当前工作版本使用的图
- `tables/`：当前工作版本使用的表
- `snippets/`：当前工作版本使用的代码片段

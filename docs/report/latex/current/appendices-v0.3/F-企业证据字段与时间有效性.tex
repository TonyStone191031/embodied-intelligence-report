\chapter{企业证据字段与时间有效性}
\label{app:company-evidence-fields}

企业记录采用以下最小字段：主体与产品版本、任务单元、客户/场地、证据日期、模型与数据、本体和末端、运行分母、人工角色、故障/安全、制造与供应链、运维、合同类型、信息来源、独立交叉来源和不可外推边界。

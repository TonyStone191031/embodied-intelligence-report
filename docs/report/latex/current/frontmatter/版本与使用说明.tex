\chapter*{版本与使用说明}
\addcontentsline{toc}{chapter}{版本与使用说明}

本目录是具身智能行业和技术发展调研报告从 v0.2 起的 LaTeX 正文工作区。v0.1 保留为 Markdown 冻结版本，不在本目录中回写或重构。

\section*{版本边界}
\addcontentsline{toc}{section}{版本边界}
本工作区当前对应 \ReportVersion。日常正文编辑只允许修改本目录下的 LaTeX 源文件；规划文件、研究笔记、论文卡、企业卡和数据资产仍按项目工作规范维护。准备冻结时，应将完整工作区复制到对应的 \texttt{docs/report/versions/<version>/latex/} 目录，再从冻结源生成发布 PDF。

\section*{阅读与构建}
\addcontentsline{toc}{section}{阅读与构建}
报告正文按章节拆分，由 \texttt{main.tex} 装配。公式直接使用 LaTeX 数学环境，长表格使用可分页表格环境，图表使用版本工作区或项目公共资产。构建中间文件统一放在 \texttt{tmp/latex-build/<version>/}，最终 PDF 写入 \texttt{output/exports/<version>/}。

\section*{交付定位}
\addcontentsline{toc}{section}{交付定位}
从 v0.2 起，PDF 是主要交付件；Word 仅作为可选兼容产物，不承诺复杂公式、浮动图表和长表格与 PDF 完全一致。

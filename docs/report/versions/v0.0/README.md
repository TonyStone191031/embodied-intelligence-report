# v0.0 冻结版本说明

## 版本状态
本目录为《具身智能行业和技术发展调研报告-v0.0》的冻结版本。

冻结日期：`2026-07-04`

## 目录内容

1. 章节正文：冻结自 `docs/report/current/` 的分章 Markdown 正文
2. `tables/`：本版本引用或配套的章节表格
3. `figures/`：本版本专用图表目录
4. `snippets/`：本版本专用代码片段目录

## 版本定位

1. `v0.0` 的目标是形成一版可持续维护的研究型长篇报告初稿，而不是终稿
2. 本版本重点完成了整体结构搭建、章节级研究正文铺开、核心技术主干的公式/伪代码/工程分析补充，以及企业/产业/争议框架的初步成型
3. 本版本默认仍有继续加密空间，尤其是在更多图表资产、更多版本化研究卡片、以及个别章节进一步增厚方面

## 与 `current/` 的关系

1. 本目录内容默认不直接改写
2. 后续 `v0.1`、`v0.2` 等版本继续在 `docs/report/current/` 中增量推进
3. 若需追溯 `v0.0` 的交付口径、章节内容或表格状态，应以本目录为准

## 备注

1. 可复用公共素材继续保留在仓库公共目录，例如 `assets/`、`research/`、`data/processed/`
2. 若后续新增仅服务于某一版本的专用图表或代码片段，应优先固化到对应版本目录下

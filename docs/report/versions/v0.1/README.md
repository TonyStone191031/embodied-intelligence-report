冻结日期：`2026-07-11`

## 目录内容

1. 章节正文：由 `docs/report/current/` 在切换到 `v0.2` 前完整复制。

3. 本次收口不重新生成 Word/PDF；导出链路保持独立维护，需发布时再由冻结版统一生成。

## 与 current 的关系

1. `docs/report/current/` 已进入 `v0.2`。
2. 后续正文、图表和案例深化只在 `current/` 进行。

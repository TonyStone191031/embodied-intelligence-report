# v0.1 版本清单

## 边界

- 报告版本：`v0.1`
- 冻结日期：`2026-07-11`
- 上一冻结版本：`v0.0`
- 后续工作版本：`v0.2`，位于 `docs/report/current/`
- 冻结性质：Markdown 源稿、章节表格与版本目录说明的可追溯快照

## 包含内容

1. `00-27` 分章 Markdown 正文及入口文件。
2. `tables/` 中与章节对应的表格源文件。
3. 空或版本专用的 `figures/`、`snippets/` 目录结构。

## 不包含内容

1. 重新导出的 Word/PDF 发布物；本次不以导出质量作为冻结阻塞项。
2. 公共可复用素材的副本；这些素材继续由 `assets/`、`research/` 与 `data/` 统一维护。

## 可追溯性规则

1. 冻结后不直接修改本目录的正文或表格。
2. 本版本的活跃规划快照见 `docs/planning/archive/v0.1/`。
3. 任何后续判断、图表或案例的修订先进入 `docs/report/current/`，并在下一次版本冻结时形成新快照。
4. 本目录中的章节表格链接固定指向 `docs/report/versions/v0.1/tables/`，以便复用导出器时嵌入同版本表格，而不误取 `current/` 的后续内容。

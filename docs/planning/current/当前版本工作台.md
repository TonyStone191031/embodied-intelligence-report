## 2. 启动原因、约束与目标

v0.2 已建立 LaTeX 主源和可复现 PDF 链路，但约 58.8 万字符被拆成 27 个部分、122 个数字章节和约 602 个标题单元；正文正式 `\\cite` 为 0，引用库仅 2 条，约 458 个 URL 多以裸链接替代引用；约 26 幅图、25 张表和 99 个代码环境又以概念图和无版本教学骨架为主。基础部分缺少推导、算例、真实实现、实验结果与失败条件，开头、结尾和“为什么重要”段落重复且模板化。因此 v0.3 将 v0.2 作为素材库和冻结基线，不采用逐段补丁式扩写。

第一性约束：从定义与数学原理走到算法、实现和工程边界；事实、实验和判断可追溯；概念只有一个主讲位置；评价位于原理、实现和证据之后；篇幅由完整知识链而非标题和重复总结产生。

本版目标：

1. 将 27 部分重构为 10 个主题部分、38 章、独立 References 与 6 个附录。
2. 将原第三至第十二部分重写成机器人数学与控制、学习型机器人、多模态基础、系统架构和 VLA 的连续教材主干。
3. 建立章节证据包、论断矩阵、图表来源、代码 commit 和复现索引。
4. 使用 `biblatex`/Biber 正式引用，裸 URL 不代替参考文献。
5. 用完整系统案例替代场景名录和企业宣传式简介。
6. 删除版本维护教程、重复“为什么重要”、空泛结论和无证据趋势判断。

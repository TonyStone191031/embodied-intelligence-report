## 1. 成书定位

v0.3 是面向研究生入门读者的教材型技术综述。默认读者具备线性代数、概率和 Python 基础，但不预设机器人学、控制、强化学习或多模态模型的系统训练。

正文目标为约 270–315 页；References、术语和复现附录约 30–45 页；成书目标为约 300–360 页。技术内容约占 80%，应用、产业、政策约占 20%。

本版不再追求“所有热点各占一个小节”，而是按知识依赖和系统接口组织内容。每个概念只有一个主讲位置；后文需要调用时只做交叉引用。

## 2. 章节公共协议

1. 每章开头只写学习目标、前置知识和本章问题，不写泛化行业背景。
2. 原理讲解必须给出符号、假设、推导或可计算关系；公式后解释数值意义和失效条件。
3. 方法讲解必须落到算法步骤、官方实现入口或明确标注的教学伪代码。
4. 模型综述必须分开描述架构、数据、目标函数、实验结果、局限和作者判断。
5. 每章只保留一个“知识小结”和一个“综述判断”；二者职责不同，不重复正文。
6. 不为不足两个实质段落的内容单设标题；列表只有在比较或流程确有结构时使用。
7. 企业、产品和政策材料不得承担学术原理的证据职责。

## 第一部分 问题定义、历史与研究方法

### 第 1 章 研究对象、读者与证据方法

1. 给出具身智能的可操作定义以及与机器人学、自动化、智能体、自动驾驶的边界。
2. 明确本书回答的技术问题、产业问题和不回答的问题。
3. 定义论文、标准、官方代码、技术文档、统计、企业材料和媒体材料的证据等级。
4. 定义事实、推断、评价及不确定性写法。
5. 给出 References、论文卡、代码锚点和图表授权的使用方式。
6. 不进入正文的内容：版本维护教程、仓库操作指南、长期更新工作流。

### 第 2 章 从经典机器人到基础模型：范式与分类坐标

1. 工业自动化、感知—规划—控制、学习型机器人和基础模型阶段的连续关系。

3. 解释“模型能力”为什么必须放回任务与身体约束中比较。
4. 用一张范式演进表区分问题设定、数据来源、接口和评测，而不是写模型年表。

### 第 3 章 具身系统的公共架构与核心矛盾

1. 感知、状态估计、世界表征、任务规划、技能、控制、安全与数据闭环的公共栈。

## 第二部分 机器人数学、运动与控制基础

### 第 4 章 数学与计算预备知识

1. 向量、矩阵、范数、特征值与条件数，只讲机器人后续会使用的部分。
2. 概率分布、贝叶斯更新、最大似然与不确定性传播。
3. 梯度、约束优化、拉格朗日乘子和数值迭代。
4. 用 Python/NumPy 小算例统一符号与数值约定。

### 第 5 章 位姿、李群、运动学与雅可比

1. 坐标系、齐次变换、旋转矩阵、四元数及各自失效模式。
2. (SO(3))、(SE(3))、指数映射和对数映射的工程含义。
3. 正运动学、逆运动学、速度运动学、雅可比、奇异性和阻尼伪逆。
4. 给出机械臂位姿链算例和官方机器人库的实现入口。
5. 说明高层动作表示如何映射到任务空间与关节空间。

### 第 6 章 动力学、接触与柔顺控制

1. 刚体动力学方程、正/逆动力学与参数辨识。
2. 接触约束、摩擦锥、互补条件和混合动力学。
3. 力/位混合控制、阻抗控制、导纳控制与任务空间控制。
4. 完整推导一个柔顺插装算例，连接观测、控制、阈值和回退。
5. 引用原始论文、现代开源实现和机器人厂商控制接口；代码片段必须固定 commit。
6. 区分控制器能解决的问题与对象感知、几何建模和任务恢复不能被替代的部分。

### 第 7 章 运动规划、轨迹优化与最优控制

1. 构型空间、碰撞检测、图搜索与采样规划。
2. 轨迹优化、时间参数化和约束处理。
3. PID、状态空间、LQR/LQG、MPC 的共同问题设定和差异。
4. 用一个受约束运动任务比较采样规划、轨迹优化和 MPC 的接口。
5. 解释学习策略与规划器/控制器的组合位置，不重复 VLA 内容。

### 第 8 章 状态估计、传感器融合与 SLAM

1. 状态、观测、可观测性、噪声和滤波问题设定。
2. 卡尔曼滤波、EKF/UKF、粒子滤波的推导与适用条件。
3. 视觉里程计、激光/视觉 SLAM、多传感器融合与语义地图。
4. 时间同步、外参漂移、延迟和状态新鲜度如何影响控制。
5. 用一个移动操作案例连接估计误差、规划失效和安全回退。

# 当前工作清单

> 当前工作版：`v0.2`

> 作用：本文件是当前活跃轮次的执行级待办清单。每次新批次必须先在这里登记范围、目标、资产与状态，再改动正文。

## 使用规则

1. `planned` 的任务必须完成章节级预审和资料定位后，才可转为 `in_progress`。
2. `in_progress` 的任务必须同时维护正文、研究底稿和质量扫描记录。
3. 任务完成前必须核对：章节职责、事实/推断/评价分层、引用、图表源文件和资产路径。
4. 需要调整章节职责时，先更新 `具身智能报告-当前详细大纲.md`，并等待用户确认。

## 状态标记

- `planned`：已确定目标，尚未开始章节级研究
- `in_progress`：已完成前置预审，正在研究或写作
- `completed`：正文、研究底稿和质量检查均已完成
- `blocked`：需要外部信息或用户决策

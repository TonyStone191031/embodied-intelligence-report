# 当前工作清单

> 当前工作版：`v0.1`
>
> 作用：本文件是当前活跃轮次的“执行级待办清单”，用于把下一批正文工作拆成可跟踪任务，并明确每项任务对应的章节位置。

## 使用规则

1. 每次开启新的正文批次前，先更新本清单，再动笔。
2. 每项任务都要写明：
   - 任务目标
   - 对应章节
   - 具体补强点
   - 当前状态
3. 若某项工作会改变章节职责或结构，先回写 `具身智能报告-当前详细大纲.md`，再更新本清单。
4. 若某项工作只属于当前轮次执行安排，可在版本冻结后随对应规划一起归档。

## 状态标记

- `planned`：已确认待做，但尚未动笔
- `in_progress`：已开始推进
- `completed`：本轮目标已完成
- `blocked`：需要额外决策或外部条件

## 章号回调说明

自本轮新增 `85. RoboCup、研究竞赛与具身评测共同体` 起，原“第十八部分”及其后的内部章号整体顺延 `+1`。历史批次中已经完成的旧章号记录保留为当时执行快照；后续一律以 `docs/planning/current/具身智能报告-当前详细大纲.md` 的当前章号为准。

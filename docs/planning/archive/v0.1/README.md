# v0.1 规划归档说明

本目录保存 `v0.1` 在实际推进中使用的活跃详细大纲、修订计划与工作清单快照。

## 归档范围

1. `具身智能报告-当前详细大纲.md`：`v0.1` 冻结时的章节职责与编号口径。
2. `当前版本修订计划.md`：`v0.1` 的语言清理、结构回调和正文深化记录。
3. `当前工作清单.md`：截至冻结时已完成的执行批次与任务状态。

## 使用规则

1. 这些文件用于追溯 `v0.1` 的写作决策，不再作为活跃版本的直接执行入口。
2. 当前工作应从 `docs/planning/项目工作规范.md`、`docs/planning/README.md` 以及 `docs/planning/current/` 的 `v0.2` 文件开始。
3. 若需要解释 `v0.1` 中某项判断为何出现，应同时查看本目录、`docs/report/versions/v0.1/` 与对应的 `research/notes/v0.1-*` 底稿。

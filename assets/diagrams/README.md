# diagrams 目录说明

## 用途

`assets/diagrams/` 存放可复用的图示源文件，当前以 mermaid 为主。

这些文件的定位不是一次性插图，而是：

1. 服务 `docs/report/current/` 的正文图示
2. 为后续版本复用提供源文件
3. 为后续脚本化或重绘提供结构骨架

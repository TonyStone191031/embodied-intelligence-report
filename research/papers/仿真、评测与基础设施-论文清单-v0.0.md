# 仿真、评测与基础设施论文清单 v0.0

更新时间：2026-07-02

## 1. 用途

本清单服务于 `14-仿真、数字孪生与评测基础设施.md`。它关注的不是单一模型性能，而是支撑研究可复现、可对比、可迁移的基础设施路线。

## 2. 当前主线划分

### 2.1 仿真平台与数字孪生

这一线关注 Isaac Sim、Gazebo、MuJoCo、Habitat 等平台如何支持训练、评测和任务脚本化。

### 2.2 sim2real 迁移

这一线关注领域随机化、域自适应、接触误差与传感器偏差如何影响真实部署。

### 2.3 benchmark 与评测协议

这一线关注评测指标到底奖励什么、哪些 benchmark 真正能区分部署能力、哪些只能反映论文技巧。

## 3. 更新规则

后续补这条线时，优先判断：

1. 新工作是在提升仿真逼真度，还是提升评测可用性。
2. 它是否改变 `14` 章对指标、迁移和复现性的判断。
3. 是否需要同步回写 `data/processed/评测指标框架-v0.0.csv`。

## 4. 后续待补对象

1. 更系统的 sim2real 代表工作
2. 新一代机器人 benchmark 与评测协议
3. 仿真数据工厂或大规模合成数据基础设施论文

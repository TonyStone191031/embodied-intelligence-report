# GR00T N1 论文卡 v0.0

更新时间：2026-07-01

## 1. 基本信息

- 论文名称：GR00T N1: An Open Foundation Model for Generalist Humanoid Robots
- 年份：2025
- 所属主线：平台型基础模型 / humanoid foundation model
- 对应章节：11、18、21

## 2. 问题设定

GR00T N1 的核心问题是：能否为通用人形机器人建立一个可扩展、可迁移、带有平台生态支撑的基础模型，使模型、仿真、合成数据与部署接口形成闭环。

## 3. 输入输出接口

- 输入：视觉、语言、机器人状态与历史上下文
- 输出：动作生成与执行控制接口
- 接口类型：平台型 VLA / dual-system architecture

## 4. 数据与训练

- 数据来源：真实轨迹、人类视频、仿真或合成数据
- 训练方式：异构数据混合训练
- 评测方式：多任务泛化、跨平台适配与人形任务表现

## 5. 论文在谱系中的角色

GR00T N1 的谱系意义，不只是“又一个 VLA”，而是把模型放回更完整的平台生态中理解：它和 Isaac、仿真、数据生成、边缘部署共同构成一个行业基础设施提案。

## 6. 工程与部署含义

1. 有助于理解为什么平台公司会在具身智能中拥有特殊地位。
2. 模型本身的能力评估，必须和仿真、数据与部署栈一起看。

## 7. 局限与后续问题

1. 平台型叙事很强，但单模型是否能真正成为事实标准仍需观察。
2. 跨本体适配、实时性和客户现场连续运行仍是独立考题。

## 8. 后续跟踪点

1. 开放程度是否继续提高。
2. 是否有更多合作伙伴在真实系统中复用。
3. 与 DeepMind、Figure、OpenVLA 路线的边界是否继续清晰化。

# 规划与具身推理论文清单 v0.0

更新时间：2026-07-02

## 1. 用途

本清单服务于 `12-规划、推理与决策的新路线.md` 与 `18-代表性论文与模型谱系梳理.md`。它的目标是把“语言规划”“程序化中间表示”“外部记忆/技能积累”这些容易混在一起的路线拆开。

## 2. 当前主线划分

### 2.1 语言规划与可供性约束

1. [SayCan 论文卡](D:/Projects/embodied-intelligence-report/research/papers/SayCan-论文卡-v0.0.md)

这条线的关键是：语言模型不能脱离机器人当前可执行技能集合单独规划。

### 2.2 程序化中间表示

1. [Code as Policies 论文卡](D:/Projects/embodied-intelligence-report/research/papers/Code-as-Policies-论文卡-v0.0.md)

这条线的关键是：把语言输出约束成可执行程序接口，增强组合性、可解释性和规则约束能力。

### 2.3 外部记忆与技能积累

1. Voyager

这条线更偏 agentic memory 和 open-ended skill accumulation，提醒我们长时程任务并不一定只能靠单轮推理完成。

## 3. 更新规则

后续新增规划类论文时，优先判断：

1. 它是在提升高层任务分解，还是在提升可执行性约束。
2. 它是纯语言推理，还是依赖技能库、代码接口或外部记忆。
3. 它是否改变第 12 章对“大脑-小脑分层”必要性的判断。

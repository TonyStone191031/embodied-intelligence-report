# companies 目录说明

## 用途

`research/companies/` 用于存放企业级长期跟踪材料，而不是正式正文。

它主要服务三类工作：

1. 企业专题章节的持续更新
2. 季度信号记录与判断回调
3. 企业之间的横向对照沉淀

## 当前建议的使用方式

后续新增企业研究材料时，优先按以下两类文件沉淀：

1. 企业长期跟踪卡
   - 适合记录较稳定信息
   - 包括本体路线、模型路线、场景、制造与交付方式、主要风险

2. 企业季度更新日志
   - 适合记录高频变动
   - 包括融资、客户、产品、部署、事故、合作、团队与政策信号

## 建议命名

1. 长期跟踪卡：
   - `企业名-长期跟踪卡-v0.0.md`

2. 季度更新日志：
   - `企业名-季度更新日志-v0.0.md`

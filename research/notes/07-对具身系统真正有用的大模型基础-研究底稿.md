# 07 对具身系统真正有用的大模型基础 - 研究底稿

## 1. 本章职责

本章不是通用 LLM / VLM 综述，而是回答三个更窄的问题：

1. 大模型的哪些能力能够真实进入具身系统主链路
2. 这些能力进入机器人后会受到哪些新的物理、时延与执行约束
3. 哪些常见叙事在机器人场景中应被主动降温

本章与相邻章节的边界：

1. 不展开 VLA 具体模型谱系，留给第 11 章
2. 不展开系统分层接口，留给第 08 章
3. 不展开数据闭环与训练工程，留给第 13 章

## 2. 本章要回答的核心问题

1. 为什么 Transformer 序列接口对机器人有吸引力
2. 为什么动作序列不能简单类比文本序列
3. 为什么语言是高层任务接口而不是低层执行接口
4. 为什么检索、记忆、分解与纠错在具身系统里比“纯参数智能”更重要
5. 哪些流行能力不应在机器人里被高估

## 4. 本章应补强的论证点

1. “长上下文”应被解释为任务历史压缩与状态新鲜度权衡，而不是单纯 token 数增长
2. “语言接口”应被解释为高层目标压缩接口，而不是执行细节承载接口
3. “常识”应被解释为先验排序器，而不是现场事实
4. “动作输出”必须明确落到控制频率、约束、稳定性与安全问题

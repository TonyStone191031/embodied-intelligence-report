# 12 规划、推理与决策的新路线 - 研究底稿

## 1. 本章职责

本章要把“高层推理”从抽象大模型能力，落成具身系统中的任务结构化与技能组织问题。

## 2. 核心问题

1. 语言规划器究竟输出什么才算可执行
2. 行为树、HTN、程序化中间表示各自适合什么角色
3. 大小脑协同里的时间尺度和接口应该怎么划分
4. 外部记忆、多智能体协同与持续规划如何进入真实系统

## 3. 关键资料

1. SayCan
2. Code as Policies
3. Voyager
4. Behavior Trees in Robotics and AI
5. 相关 HTN / 程序化代理 / 神经符号机器人工作

## 4. 本章应补强的论证点

1. 指令歧义消解必须和状态、记忆、技能库绑在一起
2. 中间表示不是装饰，而是可验证性的主要来源
3. 推理 token 增加不等于真实系统更强
4. 多机器人协作的难点在通信与耦合失败传播

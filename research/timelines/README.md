# timelines 目录说明

## 用途

`research/timelines/` 用于存放与时间线相关的辅助材料，包括：

1. 关键论文时间线
2. 企业事件时间线
3. 产业与政策信号时间线
4. 报告版本更新节律相关材料

## 当前阶段

当前阶段已先在 `data/processed/论文谱系时间线-v0.0.csv` 中建立第一版论文时间线结构化输入，用于支撑：

1. 第 18 章后续时间线重绘
2. 论文谱系的季度增量更新
3. 从长文本中抽离“年份 - 路线 - 角色”三元信息

后续若要扩展：

1. 优先继续补年份、路线、章节映射
2. 再考虑增加论文之间的前后继承关系字段
3. 若形成更复杂图谱，再转入 `code/` 做脚本化生成

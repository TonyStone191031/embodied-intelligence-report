# data/processed 目录说明

## 用途

`data/processed/` 用于存放已经过字段整理、可直接服务正文、表格或后续脚本化图表生成的结构化数据。

这些文件不是原始采集资料，而是本项目内部用于：

1. 固化分析框架
2. 支撑正文中的表格
3. 为后续 `code/` 下的批量制图脚本提供输入

## 使用原则

1. 新增结构化数据时，优先说明它服务哪个章节
2. 字段命名尽量稳定，便于后续脚本直接读取
3. 若同一框架既有 `.csv` 又有正文表格，应保持字段语义一致
